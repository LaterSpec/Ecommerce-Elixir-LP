# test

Carpeta de pruebas automatizadas.

## Archivos

### `test_helper.exs`
Configuración inicial de ExUnit:
- Inicia el framework de testing
- Configura el sandbox de base de datos en modo manual

## Subcarpetas

### `mi_tienda_web_web/`
Tests de la capa web (controllers, views).

### `support/`
Módulos de soporte para tests.

## Ejecutar tests
```bash
mix test
```
