# controllers

Tests de controladores.

## Archivos

### `error_html_test.exs`
Tests para renderizado de errores HTML (404, 500).

### `error_json_test.exs`
Tests para renderizado de errores JSON.

### `page_controller_test.exs`
Tests del controlador de páginas estáticas.

## Ejecutar solo estos tests
```bash
mix test test/mi_tienda_web_web/controllers/
```
