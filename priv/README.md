# priv

Archivos privados de la aplicación (no expuestos públicamente).

## Subcarpetas

### `gettext/`
Archivos de internacionalización (traducciones). Contiene:
- `errors.pot` - Template de mensajes de error
- `en/LC_MESSAGES/errors.po` - Traducciones al inglés

### `repo/`
Archivos relacionados con la base de datos (seeds, migraciones).

### `static/`
Archivos estáticos servidos directamente:
- `robots.txt` - Configuración para buscadores
- `assets/` - CSS y JS compilados
- `images/` - Imágenes de la aplicación
