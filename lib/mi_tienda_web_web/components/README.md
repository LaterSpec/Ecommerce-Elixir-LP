# components

Componentes reutilizables de la interfaz.

## Archivos

### `core_components.ex`
Componentes base de Phoenix:
- Botones, inputs, formularios
- Modales, flash messages
- Tablas, listas
- Iconos y helpers de UI

### `layouts.ex`
Módulo que importa los layouts de la aplicación.

## Subcarpeta

### `layouts/`
- `root.html.heex` - Layout HTML principal (head, body, scripts)
